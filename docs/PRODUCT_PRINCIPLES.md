# ZenixUI Product Principles

These principles guide the core product decisions of ZenixUI. While Design Principles focus on visual quality, these Product Principles focus on the developer experience and platform philosophy.

1. **Every feature must save developers time.**
2. **Every screen should explain itself.**
3. **Users should not need documentation.**
4. **Every copied component should work.**
5. **Generated code should be editable.**
6. **Never lock users into ZenixUI.**
7. **No runtime required.**
8. **Everything should feel premium.**
9. **Customization before configuration.**
10. **Developer happiness first.**
