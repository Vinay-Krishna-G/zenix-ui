# Hero Block

## Purpose
The primary marketing section. Placed at the top of an experience to communicate value proposition immediately. Should immediately communicate **Premium SaaS**.

## Variants
- `split`: Image or illustration on one side, text on the other.
- `centered`: Text centered, large typography, radial gradients.

## Editable Fields
- `title` (text)
- `subtitle` (textarea)
- `primaryButtonText` (text)
- `secondaryButtonText` (text)
- `image` (text, url) (Split variant only)

## Responsive Notes
- Stacks vertically on mobile (`<1024px`), expands to side-by-side on desktop.
- Typography scales fluidly.

## Accessibility
- Color contrast AA compliant (slate-900 on white).
- Button focus states use `ring-2 ring-slate-900 ring-offset-2`.
