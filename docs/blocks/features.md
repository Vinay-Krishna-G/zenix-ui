# Features Block

## Purpose
Highlights the key benefits of the product. Should immediately communicate **Capability and Value**.

## Variants
- `grid`: A robust, multi-column grid of feature cards.

## Editable Fields
- `label` (text)
- `title` (text)
- `subtitle` (textarea)
- `items` (array)
  - `title` (text)
  - `description` (textarea)
  - `icon` (text)

## Responsive Notes
- Grids shift from 1 column on mobile, to 2 on tablet, to 3 on desktop.
- Subtle hover interactions designed not to clutter the mobile touch experience.

## Accessibility
- Icon containers have sufficient contrast against the surface.
- Clear structural hierarchy.
