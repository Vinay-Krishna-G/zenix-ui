# Navbar Block

## Purpose
Global navigation header. Should immediately communicate **Clarity and Quality**.

## Variants
- `business`: Professional header with logo, standard links, and right-aligned CTA/Login.

## Editable Fields
- `links` (array of label/url)
- `showLogin` (boolean)
- `loginText` (text)
- `ctaText` (text)

## Responsive Notes
- Desktop: Horizontal links.
- Mobile: Hamburger menu that toggles an expanded dropdown below the header.

## Accessibility
- Keyboard navigable menu and CTA buttons.
- High contrast text (`text-slate-600` for inactive, `text-slate-900` for hover/active).
